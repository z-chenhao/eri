# Writing and material delivery

1. Infer or establish purpose, audience, channel, desired reaction, factual inputs, length, tone, and non-negotiable claims.
2. Separate facts, user opinions, tentative claims, and missing evidence. Never invent numbers, quotations, experience, commitments, or citations.
3. Build the argument and information order before polishing sentences. Preserve the user's recognizable voice unless explicitly asked for a different persona.
4. Produce one recommended ready-to-use version. Offer alternatives only when a real strategic choice exists, and explain the tradeoff briefly.
5. Check names, dates, amounts, links, attachments, requested action, and contradictions. For external sending, show the exact final recipient/body/attachments at the confirmation boundary.
6. Deliver long material as a sensibly named file with a short summary in conversation.
