# Research and decision support

1. Turn the decision into hard constraints, soft preferences, evaluation dimensions, time horizon, downside, and success criteria.
2. Expand the candidate set before ranking. Search multiple query formulations and prefer primary, current sources. Record event date separately from publication date.
3. Treat copied reporting as one source group. Cross-check decisive claims and preserve meaningful conflict or uncertainty.
4. Eliminate infeasible options, then compare the remaining Pareto set across value, total cost, time, reversibility, risk, and fit with governed user preferences.
5. Return a few high-ROI candidates, an explicit recommendation, the strongest objection to it, and what new fact would change the decision.
6. If the decision enables action, prepare the next step but stop at the correct confirmation, approval, authentication, or payment boundary.
